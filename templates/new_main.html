What is MagmaChain?

MagmaChain is a REST API for screenshotting websites. The service is completely free and will continue to get support. Here is an
example
 for the discord.py@rewrite library: 
            
from discord.ext import commands
import aiohttp

@bot.command()
@commands.cooldown(1, 5, commands.BucketType.guild)
async def ss(ctx, *, url: str):
  async with ctx.typing(), aiohttp.ClientSession() as session:
      screener = "http://magmachain.herokuapp.com/api/v1"
      async with session.post(screener, headers=dict(website=url)) as r:
          website = (await r.json())["snapshot"]
          await ctx.send(embed=discord.Embed(color=discord.Color.blurple()).set_image(url=website))
